# ectg-opt

ECTG-environment setup for opt. Uses the chef library/wrapper pattern and includes [ucla/chef-opt](https://github.com/ucla/chef-opt).

## Supported Platforms

CentOS6

## Usage

### ectg-opt::default

Include `ectg-opt` in your node's `run_list`.
